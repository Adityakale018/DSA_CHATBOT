An AI-powered chatbot built using Google's Gemini API that answers
Data Structures and Algorithms questions in a simple and beginner-friendly way.

## Features
- Answers only DSA-related questions
- Simple explanations for beginners
- Secure API key handling using environment variables
- Clean and professional Git workflow

## Tech Stack
- JavaScript
- Google Gemini API
- HTML & CSS
- Node.js

## Security
- API keys are stored in `.env` file
- `.env` is excluded using `.gitignore`

## How to Run
1. Clone the repo
2. Install dependencies: `npm install`
3. Create `.env` file and add your Gemini API key
4. Run the app
